Help & support

How to connect bots?

1. Open Telegram app and chat with @BotFather.
2. Send /newbot command and name your bot.
3. Copy the Bot API Token key and paste it via Add Bot here.
4. Add your bot to a group chat or message it, then click **Load chats**.

How to browse Folders?

When uploading files, simply specify a folder tag name (e.g. Travel 2026). The system will automatically build premium folders and organize your dashboard hierarchy.
